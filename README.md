# Student Grade Calculator

## Overview
This project is a Java Maven web application developed for the ASSDX4A DevOps Practical Assessment.

The application calculates:
- Final marks
- Grades
- Supplementary eligibility
- Pass rates
- Class averages

## Technologies Used
- Java 11
- Maven
- JUnit
- GitHub Actions
- Jenkins
- Docker
- Tomcat
- Render.com

## CI/CD Pipeline
The project includes:
- GitHub Actions CI pipeline
- Jenkins pipeline
- Docker containerisation
- Render deployment workflow
